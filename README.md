# Tic-Tac-Toe in Java

## Overview
A classic console-based Tic-Tac-Toe game developed in Java. This project focuses on implementing game loop logic, handling user input, and managing multi-dimensional arrays to represent the game board.

## Technical Details
* **Language:** Java
* **Concepts Applied:** 2D Arrays, game loops (`while`), conditional flow, and algorithm design for win validation.
* **Features:**
  * Interactive turn-based gameplay.
  * Real-time board updating and rendering.
